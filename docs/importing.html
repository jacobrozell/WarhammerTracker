Importing CSV

The Muster Roll uses two CSV files — one for armies, one for paints. They are imported separately.

First-time setup

  1. Open the app
  2. On the Armies tab, click Army Template or use the sample warhammer_armies.csv in the repo
  3. Fill in your collection (spreadsheet or text editor)
  4. Click Import Armies CSV and select your file
  5. Repeat on the Paints tab with warhammer_paint_inventory.csv

You can also drag a CSV onto the page or the empty-state drop zone.

Column reference

Full schema: SCHEMA.md

  • Armies (required): Game, Faction, Army, Unit
  • Paints (required): Name

Tips

  • Headers are case-insensitive
  • UTF-8 with BOM is supported; TSV and semicolon files are auto-detected
  • Fields with commas must be quoted: "Notes, with comma"
  • Import replaces the current tab's data unless you use Append
  • Unknown painting states fall back to the first pipeline stage with a warning

Export / backup

  • Export Armies / Export Paints — CSV backup compatible with re-import
  • Full backup — JSON file including settings and pipeline

Troubleshooting

ProblemFix

"Missing required columns"Compare headers to SCHEMA.md
